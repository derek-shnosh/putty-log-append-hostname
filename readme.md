PuTTY cannot automatically parse the hostname from an SSH session to be used within the log file name variables, preventing the use of the device's hostname in the log file for any sessions established via IP.

This is to be used with PuTTY .log files generated by "Gather Scripts" containing the output of `show running-configuration`.

This script will parse all log files in the specified directory for a line that starts with "hostname" to capture the hostname and append it to the front of
the filename.

Usage: `python3 ./append_hostname.py ./logfiles/`
